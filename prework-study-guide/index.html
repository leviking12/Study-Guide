Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is important to build a good foundation for your website.
  • Use semantic elements as much as possible

CSS

  • There are multiple ways to style a webpage using CSS
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets
  • CSS is used to design/customize certain elements on the webpage

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is a version control software
  • Git makes collaborating on projects easy

JavaScript

  • JavaScript is used to add functionality to a website
  • JavaScript is a set of code that runs in the background of a webpage
